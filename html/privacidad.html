Recogida y uso de sus datos personales

Tipos de datos recogidos

Datos personales

Mientras utiliza nuestro Servicio, es posible que os pidamos que nos proporcione cierta información de identificación personal que se puede utilizar para contactar o identificarse. La información de identificación personal puede incluir, pero no limitada a:

  • Dirección de correo electrónico

  • Nombre y apellidos

  • Datos de uso

Datos de uso

Los datos de uso se recogen automáticamente cuando se utiliza el Servicio.

Los datos de uso pueden incluir información como la dirección de protocolo de Internet de su dispositivo (por ejemplo, la dirección IP), el tipo de navegador, la versión del navegador, las páginas de nuestro servicio que visita, la hora y la fecha de su visita, el tiempo que pasa en estas páginas, identificadores únicos del dispositivo y otros datos de diagnóstico.

Cuando acceda al Servicio mediante un dispositivo móvil o mediante un dispositivo móvil, podemos recopilar cierta información automáticamente, incluido, entre otros, el tipo de dispositivo móvil que está utilizando, el identificador único del su dispositivo móvil, la dirección IP de su dispositivo móvil. , Su sistema operativo móvil, el tipo de navegador de Internet móvil que utiliza, identificadores únicos del dispositivo y otros datos de diagnóstico.

También podemos recopilar información que su navegador envía cada vez que visite nuestro Servicio o cuando acceda al Servicio mediante o mediante un dispositivo móvil.

Tecnologías de seguimiento y galletas

Utilizamos cookies y tecnologías de seguimiento similares para realizar un seguimiento de la actividad a nuestro servicio y almacenar cierta información. Las tecnologías de seguimiento utilizadas son balizas, etiquetas y scripts para recopilar y realizar un seguimiento de información

El uso de las galletas

Tipo de galletas que utilizamos

Las cookies pueden ser "Persistente" o "Sesión" Cookies. Las galletas persistentes permanecen en su ordenador personal o dispositivo móvil cuando os desconecte, mientras que las cookies de sesión se eliminan en cuanto cierre el navegador web.

Usamos tanto cookies de sesión como persistentes con las finalidades que se indican a continuación:

  • Cookies necesarias/esenciales

    Tipo: cookies de sesión

    Administrado por: Nosotros

    Propósito: Estas cookies son esenciales para ofrecerle servicios disponibles a través del sitio web y para permitirle utilizar algunas de sus características. Ayudan a autenticar usuarios y prevenir el uso fraudulento de las cuentas de usuario. Sin estas cookies, los servicios que ha solicitado no se pueden ofrecer, y sólo utilizamos estas cookies para ofrecerle estos servicios.

  • Galletas de funcionalidad

    Tipo: cookies persistentes

    Administrado por: Nosotros

    Propósito: estas cookies nos permiten recordar las opciones que haga cuando utilice el sitio web, tales como recordar sus datos de inicio de sesión o la preferencia de idioma. El objetivo de estas galletas es ofrecerle una experiencia más personal y evitar que tenga que volver a introducir sus preferencias cada vez que está utilizando el sitio web.

Sus elecciones con respecto a las cookies

Si prefieres evitar el uso de cookies en el sitio web, primero tendrás que desactivar el uso de cookies en tu navegador y después eliminar las cookies guardadas en tu navegador asociadas a este sitio web. Puede utilizar esta opción para impedir el uso de cookies en cualquier momento.

Si no acepta nuestras cookies, es posible que experimente algunos inconvenientes en el uso del sitio web y que algunas funciones no funcionen correctamente.

Si desea eliminar cookies o indicarlo en su navegador web que borre o rechace las cookies, visite las páginas de ayuda de su navegador web.

  • Para el navegador web Chrome, visiten esta página de Google: https://support.google.com/accounts/answer/32050

  • Para el navegador web Internet Explorer, visiten esta página de Microsoft: http ://support.microsoft.com/kb/278835

  • Para el navegador web Firefox, visite esta página de Mozilla: https://support.mozilla.org/en-US/kb/delete-cookies-remove-info-websites-stored

  • Para el navegador web Safari, visiten esta página d'Apple: https://support.apple.com/guide/safari/manage-cookies-and-website-data-sfri11471/mac

Para cualquier otro navegador web, visite las páginas web oficiales de su navegador web.
